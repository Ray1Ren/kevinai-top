# PRODUCT.md

## 产品目标

kevinai.top 是 Kevin 的长期个人主页，不是某一次评测的专题站。

- 让访客先理解 Kevin 是谁、正在做什么，以及判断与创作方式。
- 展示真正上线的产品；当前主项目是《一脚晋级》。
- 建立可持续的文章 / 公众号内容归档；首篇开发日记已有中英文版，后续内容仍需逐篇核对。
- 将评测原始网页、最终结果、公开数据与筛选证据放入独立 /lab 栏目。
- 提供 /en 英文版本；分数、用时和冻结证据不得因翻译改变。

## 数据口径

- 四项最终质量分：速度不计入；四项等权，各占 25%。
- 模型展示顺序固定为：Kimi K3、GPT-5.6 Sol、MiniMax M3。
- 宣传页最终质量分为 K3 91.0、Codex 95.0、MiniMax M3 85.0；5:45 等用时只单列展示，不折算进质量分。
- 识图榜按公众号最终稿的四维度评分；速度不计分。
- 中英文 JSON 的所有数字必须一致，翻译只能改变说明文字。
- Kimi K3 评测的中英文页面共用同一组分数与额度事实；发布门控固定为 `2026-07-19T00:00:00Z`。

## 边界与红线

- 不添加未经来源支持的履历、公司、奖项、用户量、收入、下载量或团队规模。
- 对外游戏名只写《一脚晋级》，不公开内部项目名。
- 不公开本机绝对路径、原始日志、密钥、邮箱、手机号或私有 manifest。
- 不把尚未整理好的公众号内容伪装成已发布文章。

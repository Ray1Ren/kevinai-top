# DESIGN.md

## 视觉概念

**“从想法到作品的轨道档案”**。深石墨色空间中，粒子与细线从想法汇聚成实验，再形成真正上线的作品；头像是系统中的人类判断核心。评测只作为长期档案的一条支线，不占据首页主叙事。

## 设计拨盘

- `DESIGN_VARIANCE = 9`：非对称布局，左侧文字、右侧粒子系统。
- `MOTION_INTENSITY = 9`：3D 粒子轨道 + 滚动叙事 + 微交互。
- `VISUAL_DENSITY = 5`：内容疏密适中，正文可读性优先。

## 色彩系统

- 背景：深石墨 `#141413`
- 文字：灰阶 100 ~ 400
- 单一强调色：球场绿/黄绿色 `#8ab82f`
- 品牌色仅用于 UI 强调；头像中的橙色和青色不扩散为其他 UI 配色。

## 字体

使用系统字体栈，无外部字体请求。

## 动效规则

- R3F/Three.js 独立 Canvas 负责粒子轨道主视觉。
- UI 进入/退出使用 React + CSS transition。
- 不混用 GSAP 与 Framer Motion。
- 粒子数按设备封顶：桌面 520、平板 220；手机与粗指针直接使用静态轨道。DPR 封顶 1.5。
- `prefers-reduced-motion` 或手动暂停按钮会停止自动运动。
- 复杂 3D 交互在 `pointer: coarse` 或窄屏下关闭。

## 可访问性

- 语义化 HTML。
- 可见 focus ring（`outline: 2px solid #8ab82f`）。
- 所有图片带 alt。
- 支持键盘导航。
- 提供“暂停动效”控制。
- 中英文页面使用独立可索引路径，并提供 hreflang。

## 响应式

- 首屏使用 `min-height: 100dvh`。
- 无横向溢出。
- 移动端导航为下拉菜单。
